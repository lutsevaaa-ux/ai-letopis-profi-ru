## Краткое резюме всех этапов проекта

- Сначала проект задумывался как пилот по сбору событий из нескольких Mattermost-каналов за ограниченный период.
- Затем появилась система отбора кандидатов по активным тредам и ручной редакторской фильтрации.
- После этого был собран шорт-лист карточек и единый JSON-формат для летописи.
- Дальше проект вырос в визуальную витрину в стилистике старого свитка.
- Следующим шагом появился редактор, чтобы обновлять данные без ручной правки кода.
- Финалом стала годовая версия летописи на 12 месяцев с публикацией через `docs`.
