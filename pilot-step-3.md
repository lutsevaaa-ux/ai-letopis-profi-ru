# Шаг 3. Шорт-лист лучших карточек

## Что сделано

Из `8` кандидатов первого прохода собран шорт-лист из `6` лучших карточек для пилота.

Логика отбора была такой:

- событие должно быть понятно без длинного контекста
- оно должно быть либо важным, либо очень запоминающимся
- по нему должна легко собираться визуальная карточка
- в наборе должны быть и рабочие, и культурные сюжеты

## Что вошло в шорт-лист

1. Иностранная карта и `eSIM` для зарубежных сервисов
2. Запуск первых сценариев `Профи-Стандарта`
3. Утверждение летнего выезда для хэдофиса
4. Жёсткое правило по газовому оборудованию
5. Обсуждение рекламы конкурента как внутреннего мема
6. Праздничный потешный четверг перед майскими

## Почему этот набор выглядит удачно

Он уже похож на настоящую летопись:

- есть продуктовые изменения
- есть операционные решения
- есть культурные и мемные эпизоды
- есть внешний инфоповод
- есть история про повседневный AI-быт команды

## Что дальше

Следующий логичный шаг:

- превратить эти `6` событий в единый формат карточек
- для каждой карточки собрать:
  - месяц
  - заголовок
  - короткое описание
  - мем-идею
  - визуальный промпт или концепт

После этого можно будет либо делать текстовый черновик летописи, либо сразу собирать демо-страницу.
